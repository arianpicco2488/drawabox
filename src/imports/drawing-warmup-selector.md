Design a mobile app screen for selecting a random drawing warm-up exercise.

The interface should resemble a clean digital sketchbook used for drawing practice.

Visual Style

Use a light paper-like background (#FAF7F2) to resemble drawing paper.

Use simple black linework (#222222) for text, icons, and separators.

Use a warm orange accent color (#F05A28) for interactive elements such as buttons, highlights, and the roulette pointer.

Borders should be thin and light grey (#D9D4CC).

The interface should feel minimal, instructional, and artistic, like a tool used by art students.

Typography should be clean sans-serif, with clear hierarchy:

large bold title

medium exercise titles

smaller instruction text

Use plenty of whitespace.

Screen Layout

Top section:
Title centered at the top

“Warm-Up Selector”

Below the title place a large roulette wheel component.

The roulette wheel contains segments representing drawing exercises.

Each segment contains the exercise name.

Above the wheel place a small orange pointer indicating the selected result.

Below the wheel place a large circular “SPIN” button with an orange border.

After Spinning

When the roulette stops, the wheel fades slightly and a result card appears below it.

The result card contains:

Exercise Name (bold)

Lesson Number

Short Instructions

Small sketch-style reference illustration.

Example layout of the card:

Exercise Name
Lesson #

Sketch example image

Instructions explaining what the exercise practices.
