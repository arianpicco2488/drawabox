Design a mobile app screen for browsing drawing warm-up exercises in a gallery format.

Visual Style

Use a light paper-like background (#FAF7F2) similar to drawing paper.

Use black or very dark grey text (#222222) for titles and labels.

Use thin light grey borders (#D9D4CC) for cards and separators.

Use a warm orange accent color (#F05A28) for highlights, selected elements, and interactive components.

The design should feel like a minimal digital sketchbook used for drawing practice.

Use plenty of whitespace and clean sans-serif typography.

Screen Layout

At the top of the screen display the title:

Warm-Up Gallery

Below the title show a grid of square tiles arranged in three columns.

Each tile represents a drawing warm-up exercise.

Each square tile should include:

a simple sketch-style icon or faint drawing

the exercise name

thin grey border

white or paper-colored background

Tiles should look like organized practice cards in a sketchbook.

Interaction

When a user taps a square tile, a modal window appears.

The modal contains:

Exercise Name (large title)

Lesson Number

Sketch-style reference image

Description explaining the exercise

Below the description place a button labeled “Start Drawing”.

Modal style:

white/paper background

rounded corners

thin grey border

subtle shadow
